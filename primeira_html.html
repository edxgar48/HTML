GitHowTo_Do

Os comandos para criar um novo branch são :

git checkout "nome do Branch ... exemplo dev1...dev2..."

Os comandos principais para enviar para o branch são:

  1. git add "nome do arquivo"
  2. git commit -m "comentario"
  3. git push origin "nome do branch"
